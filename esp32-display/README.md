# ESP32 PTV Departure Display

An ESP32-powered OLED display that shows live Melbourne public transport departures.

The device connects to Wi-Fi, calls the custom public transport departures API in `/ptv-api`, parses the returned JSON, and displays upcoming departures on a 128×64 SSD1306 OLED screen.

## Software Required

Install the Arduino IDE:

https://www.arduino.cc/en/software

This project was developed using the Arduino IDE with ESP32 board support.

## Arduino IDE Setup

### 1. Install ESP32 Board Support

Open Arduino IDE.

Go to:

```text
File → Preferences
```

In **Additional boards manager URLs**, add:

```text
https://espressif.github.io/arduino-esp32/package_esp32_index.json
```

Then go to:

```text
Tools → Board → Boards Manager
```

Search for:

```text
esp32
```

Install:

```text
esp32 by Espressif Systems
```

### 2. Select the ESP32 Board

Go to:

```text
Tools → Board
```

Select your ESP32 board. For many common boards, this will be:

```text
ESP32 Dev Module
```

### 3. Select the Upload Port

Connect the ESP32 via USB.

Go to:

```text
Tools → Port
```

Select the COM port for the ESP32, such as:

```text
COM4
```

## Required Arduino Libraries

Install the following libraries through the Arduino IDE Library Manager.

Go to:

```text
Sketch → Include Library → Manage Libraries...
```

Install:

```text
Adafruit GFX Library
Adafruit SSD1306
ArduinoJson
```

The following libraries are included with the ESP32 Arduino core and do not usually need separate installation:

```cpp
#include <Wire.h>
#include <WiFi.h>
#include <HTTPClient.h>
```

## Configuration

Before uploading, update the Wi-Fi credentials in the `.ino` file:

```cpp
const char* SSID = "YourWiFiName";
const char* PASSWORD = "YourWiFiPassword";
```

Then configure the stop, platform, and number of departures:

```cpp
#define STOP_ID "G1071"
#define PLATFORM "P1"
#define LIMIT 4
```

Example API URL generated by the firmware:

```text
https://esp32-ptv-display-um8x.vercel.app/departures?stop_id=G1071&platform=P1&limit=4
```

## Uploading to the ESP32

1. Open the `.ino` file in Arduino IDE.
2. Select the correct ESP32 board.
3. Select the correct COM port.
4. Click **Upload**.

## Example Output

The OLED displays the station/platform and upcoming departures, for example:

```text
Flinders Platform P1

06:46 Mooroolb (5m)
06:57 Belgrave (17m)
07:10 Lilydale (30m)
07:14 Mooroolb (34m)
```

Because the display is only 128×64 pixels, long destination names are shortened.
